# Chilid/XSolve recruitment task 2018

Front-end developer task for summer internships built in React.js.

How to use:
1) npm install
2) npm start

This code was written in ES6, SASS and namespaced BEM. Data loads from JSON via fetch call. 

Funcionality:
- pagination
- sorting
- RWD
